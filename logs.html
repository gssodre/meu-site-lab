Arquivos importantes

  • /var/log/apache2/access.log
  • /var/log/apache2/error.log

Comandos úteis

sudo tail -f /var/log/apache2/access.log
sudo tail -f /var/log/apache2/error.log

O que observar

  • quem acessou
  • qual arquivo foi pedido
  • se houve erro 404
  • data e hora do acesso
