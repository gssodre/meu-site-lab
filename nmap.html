Nmap

Enumeração inicial do servidor no laboratório.

Início Rede SSH Apache Nmap Logs Erros

Comando principal

nmap -sV 192.168.56.13

O que esse comando faz

  • descobre portas abertas
  • tenta identificar serviços
  • tenta mostrar versões

Exemplo de resultado

PORT   STATE SERVICE VERSION
22/tcp open  ssh
80/tcp open  http    Apache httpd
